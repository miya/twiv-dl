# twiv-dl
Twitterの動画をダウロードするWebApp

<img src="https://raw.githubusercontent.com/wiki/miya/twiv-dl/images/demo2.gif" width=100% alt="">
 
## Usage
1. Twitterで動画付きツイートを探す
2. ダウンロードしたいツイートの動画の上で右クリックで「動画のアドレスをコピー」をする
3. [twiv-dl](https://twivdl.appspot.com/)にアクセス
4. twiv-dlの検索窓にコピーしたアドレスを入れて検索
5. 後は煮るなり焼くなり....
